# 📊 citracer - Visualize how research ideas connect together

[![Download Citracer](https://img.shields.io/badge/Download-Citracer-blue)](https://github.com/darkbluevn1765/citracer)

Citracer helps researchers map the history of academic ideas. It scans research papers to find citations and builds a visual map. You see how one author builds on the work of another. This tool simplifies the literature review process by turning lists of papers into an interactive graph.

## 🛠 Features

*   **PDF Analysis**: The tool reads your PDF files and extracts key metadata.
*   **Graph Rendering**: It generates a visual network of your research concepts.
*   **Citation Tracking**: It maps the lineage of an idea across multiple decades.
*   **Local Processing**: Your documents stay on your computer for privacy.
*   **Search Filters**: You find specific authors or dates within the map.

## ⚙️ System Requirements

Citracer runs on Windows 10 or Windows 11. Your computer needs at least 8 gigabytes of RAM to process large batches of papers. You should have a modern processor and a stable internet connection for the initial setup.

## 🖥 Using the Interface

The interface consists of three main areas. The top menu bar contains your file controls. The left panel shows your current list of papers. The large center pane displays your citation graph.

To add papers, click the Import button in the top menu. Select the folder containing your PDF files. Citracer scans each document. It identifies the title, author, and reference list. This takes a few minutes depending on the file count. 

Once the scan finishes, the graph appears. Each node represents a paper. Lines between nodes represent citations. You click a node to see the details of a paper. You drag your mouse to move the graph around. Use your mouse scroll wheel to zoom in on specific connections.

Research often involves hundreds of documents. A spreadsheet captures facts but fails to capture structure. Citracer provides structure. When you see a dense cluster of nodes, you identify a core topic in your field. When you see a small, distant group of nodes, you identify a niche area for potential inquiry.

You export the graph as a high-resolution image for your manuscripts. You also export the underlying data as a CSV file if you prefer to sort the information in other programs.

## 🔍 Frequently Asked Questions

**Does this software send my data to a server?**
No. Citracer processes your files locally on your machine. Your research stays private.

**Which PDF formats work best?**
The tool works best with text-based PDFs. If you use scanned PDFs that consist of images, ensure your documents contain searchable text layers.

**How do I update the software?**
When a new version releases, visit the download link again. Run the installer to overwrite the old version. Your settings and processed graph data persist through this update.

**Can I run this on a laptop?**
Yes. Modern laptops handle this tool well. If you have thousands of papers, use a machine with more memory to keep the interface responsive.

**What happens if the text extraction fails?**
Citracer flags papers with unreadable text in the sidebar. You click the flag icon to manually verify the citation data. This ensures your graph remains accurate.

## 📁 Managing Your Library

Organize your PDF files into subfolders before you import them. Citracer uses these folder names to color-code your graph. This helps you visually distinguish between different projects or research themes. 

To clear your graph, go to the File menu and select Reset. This clears the workspace but keeps your original PDF files untouched. You start a fresh project with a clean slate whenever needed.

## 🛠 Troubleshooting

If the software does not open, ensure you have the latest updates for your Windows system. Check that no other citation software conflicts with the file paths used by Citracer. 

If a file fails to load, verify that the PDF is not password-protected. Citracer cannot read encrypted files. Remove the password protection before you import the document.
